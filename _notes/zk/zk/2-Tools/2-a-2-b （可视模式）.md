# Vim可视模式

## Vim可视模式基本操作

|   命令   |      用途       |
| :----: | :-----------: |
|   v    |  激活面向字符的可视模式  |
|   V    |  激活面向行的可视模式   |
| \<C-v> |  激活面向列块的可视模式  |
|   gv   |   重选上次的高亮选区   |
| **o**  | **切换高亮区的活动端** |

> [!NOTE] gv命令
> gv 命令是个有用的快捷键，它用来重选上一次由可视模式选择的文本范围。不管上个选区是面向字符的、面向行的，还是面向列块的，gv 命令都能够正确地工作。不过如果上次的选区被删除了，它可能会工作得不太正常。
